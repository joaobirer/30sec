# 30 Seconds of Chaos

A live public screen where users can pay to display content for 30 seconds.

## How it works

- **$1** — Static image (JPG, PNG, WEBP)
- **$4** — Animated GIF
- Each paid submission is added to a **global queue**
- Queue is processed in order: first paid, first shown
- Every submission appears for **30 seconds**

## Platform rules

This is a user-generated content platform.

- Illegal content is strictly prohibited
- We may remove content and block users/IPs at our discretion
- Payments purchase temporary display time only (30 seconds), not permanent hosting
- Report illegal content immediately

## Supported files

- Static: JPG, PNG, WEBP
- Animated: GIF
- Max file size: 10MB
